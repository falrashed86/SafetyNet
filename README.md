# SafetyNet

AI-based child safety monitoring system using Streamlit.

## Features
- Child App
- Parent Dashboard
- AI risk detection (LOW, MEDIUM, HIGH)
- Keyword override
- Real-time alerts
- Privacy protection

## Run locally

Install:
py -3.9 -m pip install -r requirements.txt

Run:
py -3.9 -m streamlit run dashboard/app.py
